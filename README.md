# 📊 Sales & Revenue Analysis Dashboard

## 📌 Project Overview
This project is an interactive Sales & Revenue Analysis Dashboard developed using Microsoft Power BI. It helps analyze sales performance, revenue trends, customer behavior, and product performance through interactive visualizations.

---

## 🎯 Objectives
- Analyze overall sales performance.
- Track revenue trends over time.
- Identify top-performing products.
- Compare sales across different regions.
- Support business decision-making using data.

---

## 🛠️ Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Data Visualization

---

## 📈 Dashboard Features
- Total Sales KPI
- Total Revenue KPI
- Monthly Sales Trend
- Region-wise Sales Analysis
- Product Category Performance
- Interactive Filters and Slicers
